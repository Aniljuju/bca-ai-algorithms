# BCA AI Algorithms Lab - TU

This repository contains Python implementations of various AI algorithms for BCA Semester 7 lab at Tribhuvan University (TU).

## Algorithms Included

### Search Algorithms
- **A\* Search** (`A_Star.py`)
- **AO\* Search** (`AO_Star.py`)
- **Greedy Best-First Search** (`BestFirst(Greedy).py`)
- **Iterative Deepening A\*** (`IDA_Star.py`)
- **Breadth-First Search (BFS)** (`BFS.py`)
- **Depth-First Search (DFS)** (`DFS.py`, `RecursiveDFS.py`, `IterativeDFS.py`)
- **Iterative Deepening DFS (IDDFS)** (`IDDFS.py`)

### Game / Decision Algorithms
- **MinMax** (`MinMax.py`)
- **Alpha-Beta Pruning** (`Alpha-BetaPrune.py`)

### Problem Solving
- **N-Queens Problem** (`N-Queen.py`, `N-QueenRecursive.py`)
- **Water Jug Problem** (`waterjug.py`)
